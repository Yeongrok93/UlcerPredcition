Frequently Asked Questions (FAQ)

Q1. How is the pressure injury risk calculated? +
A machine learning algorithm converts the similarity between the input feature pattern and pressure injury cases into a probability estimate.
Q2. What is the accuracy of this model? +
The model was trained on data from approximately 22,000 adult ICU patients, and achieved an accuracy of approximately 90.6% on the test dataset.
Q3. Are the entered data stored? +
No. All input values are discarded immediately after prediction and are not stored on the server.
Q4. Will the model be updated in the future? +
Yes. The model will be periodically retrained as new data become available to improve predictive performance.
