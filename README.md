Ulcer Prediction

A web application that uses machine learning to predict the likelihood of pressure injuries (ulcers) in adult intensive care unit patients.

In compliance with international reporting standards, this repository offers a web-based user interface and a lightweight Flask-based backend for transparent and repeatable research.

---

Summary

In critical care units, pressure injuries pose a serious threat to patient safety.  
In this project, a machine learning model based on regularly gathered clinical information is used to estimate the likelihood that pressure injuries would occur.

Users can enter patient data via the web interface and get an instant risk prediction.

---

Synopsis of the Model

 Adult patients in the intensive care unit  
 About 22,000 patients make up the sample size.  
 Method: Classification using supervised machine learning  
 -Performance:  
   Test precision: approximately 90.6%
 - Inputs (for instance):
   Consciousness level (RASS)
   Strength of the muscles in the lower extremities
   Body temperature (maximum/minimum)
   Frequency of daily incontinence
